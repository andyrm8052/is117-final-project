Project 2

Authored By: Andy Marmolejos

For project two our objective is to create a website with F and Z layouts using Bootstrap. In this case, this website that you are browsing at the moment is project two. Same as project one, we will be setting the website to use Gulp and BrowserSync for live updates as we make changes.

First we run "npm install" then we set-up a Gulp serve in order for us to instantly open our website updated with just one click. Second, we then clone the Bootstrap github url and work from there. Third, we then set-up our github repository with Webstorm in order for us to commit and push the project or any changes. Finally, after doing so we can set-up github with "master branch/docs folder" in order to see the live result of out website.
